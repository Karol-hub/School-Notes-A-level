- First program that runs when a computer is powered
- [[Registers|Program Counter]] points to the BIOS at each start-up

Responsible for running key tests
- Power On Self Test (POST)
	- Ensures all [[Hardware]] are correctly connected and functional
- Checking CPU clock, memory and processor is operational
- Testing for external storage devices connected to the device

Only after these checks are completed the [[Operating System (OS)]] is loaded from [[Secondary Storage]] to [[Main Memory|RAM]]
