The angle the object moves per unit time
$$\omega = \frac{2\pi}{T} = 2\pi f$$

| Symbol   | Unit        | Definition      |
| -------- | ----------- | --------------- |
| $\omega$ | $rads^{-1}$ | Angular Speed   |
| $T$      | $s$         | [[Time Period]] |
| $f$         | $s^{-1}$            | [[Frequency]]                |
