The angle an object moves through per unit time
$$\omega = \frac{v}{r}$$
or the more common form
$$v = \omega r$$

| Symbol | Unit | Definition |
| ---- | ---- | ---- |
| $\omega$ | $rads^{-1}$ | [[5 Radians\|Radians]] per second |
| $v$ | $ms^{-1}$ | [[Velocity]] |
| $r$ | $m$ | radius |
