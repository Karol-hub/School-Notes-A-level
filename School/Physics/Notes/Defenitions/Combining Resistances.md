# Series Circuit
$$R_T=R_1+R_2+R_3+...$$

| Symbol | Unit     | Definition                   |
| ------ | -------- | ---------------------------- |
| $R_T$  | $\Omega$ | Total [[Resistance]]         |
| $R_n$  | $\Omega$ | [[Resistance]] of resistor n |
# Parallel Circuit
$$\frac{1}{R_T}=\frac{1}{R_1}+\frac{1}{R_2}+\frac{1}{R_3}+...$$

| Symbol | Unit     | Definition                   |
| ------ | -------- | ---------------------------- |
| $R_T$  | $\Omega$ | Total [[Resistance]]         |
| $R_n$  | $\Omega$ | [[Resistance]] of resistor n |
