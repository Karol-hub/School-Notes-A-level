How many times an oscillation happens happens per unit time
$$f = \frac{1}{T} = \frac{\omega}{2\pi}$$
*Equation with $\omega$ is for [[Topic 13- Oscillations]]*

| Symbol | Unit     | Definition      |
| ------ | -------- | --------------- |
| $f$    | $s^{-1}$ | Frequency       |
| $T$    | $s$      | [[Time Period]] |
| $\omega$       | $rads^{-1}$         | [[Angular Speed]]                |
