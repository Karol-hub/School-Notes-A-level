A measure of how difficult it is for charge carriers to pass through a component
SI units: $kg \, m^2 s^{-3} A^{-2}$
$$R =\frac{V}{I}$$

| Symbol | Unit     | Definition  |
| ------ | -------- | ----------- |
| $R$    | $\Omega$ | Resistance  |
| $V$    | $V$      | [[Voltage]] |
| $I$    | $A$      | [[Current]] |
